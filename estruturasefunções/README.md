📚 Sistema de Cadastro de Alunos
📌 Descrição

Projeto simples em Python para prática lógica de programação.
Permite cadastrar alunos, evitar duplicações e calcular a média das notas.

⚙️ Funcionalidades
Cadastro de alunos com cada
Validação para não repetir nomes
Cálculo automático da média
Exibição de status (Aprovado/Reprovado)

🧠 Conceitos usados
Listas, conjuntos e dicionários
Funções
Condições ( if/else)
Laços ( for)

🚀 Como funciona

O sistema armazena os alunos em uma lista e usa um conjunto ( set) para evitar nomes duplicados.
Cada aluno possui nome e notas, e a média é calculada automaticamente para definir se está aprovado ou reprovado.

📊 Regra de aprovação
Média ≥ 6 → Aprovado
Média < 6 → Reprovado
